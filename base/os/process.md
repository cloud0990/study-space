```javascript
프로세스(실행 중인 프로그램)
    - 저장장치에 저장된 프로그램 파일이 메모리에 올라간다 = 실행 중이다
    - 메모리 구조, 프로세스 제어블록(PCB) 으로 나눌 수 있음
    - CPU, 메모리, 파일, I/O 장치 등 실행에 필요한 자원이 할당됨
    - 생성-준비-실행-대기-종료 중 하나의 상태로 존재
    - CPU의 스케줄링, 입출력 대기 등에 따라 준비, 실행, 대기 등으로 상태가 변화되며 동작

    * 프로세스 제어블록(PCB): 프로세스를 명시하는 다양한 내용 포함

프로세스 관리자: CPU의 할당, 회수 등 상황에 따라 프로세스의 현재 상태를 다른 상태로 변화시킴
    - 프로세스 생성 or 삭제
    - 프로세스에 CPU를 할당하기 위한 스케줄 결정
    - 각 프로세스의 상태를 관리/링전이

+) 디스패치: 특정 프로세스에 CPU를 할당하는 과정
```
